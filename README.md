Implement pow(A, B) % C. In other words, given A, B and C, Find (AB % C).

Note: The remainders on division cannot be negative. In other words, make sure the answer you return is non-negative.

Example Input

Input 1:
A = 2
B = 3
C = 3

Input 2:
A = 3
B = 3
C = 1

Example Output

Out
put 1:
2
Output 2:
0
